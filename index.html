Database

The classification, organization and versioning of core data assets across the development lifecycle.

Scan Database

A stand alone scan of database data. Analyse Column data looking for types of data, sample data and data characteristics.

Mask Database

Configure and define database masking. Assemble the rule and data definitions to define a process to mask data in a database.

Compare Database

Configure and define database comparison. The comparison can compare data across schemas, databases or using snapshots.

Subset & Clone Database

Configure and define subsetting and cloning. Subsetting extracts selected rows and tree walks across to related tables. clones of interesting data or extract bug data into development

Validate Data

Scan Databases against a defined set of data rules. Check that user defined "soft" links are valid and report on invalid data

Synthetic Data

Configure and Define database generation processes and models. Create complex interrelated data into database tables and csvs.

Modelling

Modelling to create covered sets of rich combinatorial test data.

Create Models with Solver Logic

Assemble the components to solve and model the expected results based on formulas and constraints. This model will calculate all the input values to create a specific expected result.

Complex Datasets

Assemble and define a set of data states and associated data conditions in order to build complex data sets. The data sets can then be used in models to drive data driven test automation

Files

Generate, Analyse and Mask files.

Create Messages and Files

Configure and Define file and message creation and associated models. Any structured messages can be defined with complex data generation functions. The varieties of message types can be modelled to create covered sets of messages for testingor virtualization.

Mask Files

Configure and define file masking. Assemble the rule and file definitions to define a process to mask files.

Scan Files for Data Analyzer

Scan groups of files to create meta data for analysis by Data Analyser. Look for patterns of data combinations, compare sets of characteristics and look for differences in coverage.
